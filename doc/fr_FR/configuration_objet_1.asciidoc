La configuration des Equipements "Thermostat" est accessible à partir du menu plugin :
il vous suffit de cliquer sur le bouton "ajouter" ou sur un équipement déjà présent.

La premiere étape, commune aux plugins Jeedom, consiste au réglage des paramètres genéreaux du thermostat

[NOTE]
